🔊 HOW THE ULTRASONIC SENSOR UPGRADES YOUR CAR

The HC-SR04 sends ultrasonic pulses and measures how long the echo takes to return. This tells the ESP32 exactly how far away the nearest object is — whether it's a wall, an obstacle, or the fire itself.

🛡️ FEATURE 1 — OBSTACLE AVOIDANCE

While patrolling (no fire detected), the car constantly checks distance.

If an object is closer than 15 cm and it's NOT a fire:
→ Back up → Turn right → Resume forward patrol.

Without this, the car would crash into walls and get stuck.

🔥 FEATURE 2 — SMART FIRE APPROACH

When center IR detects fire, the car approaches slowly while monitoring distance.

• < 30 cm → Start spraying water
• < 10 cm → Emergency stop (too close!)

Without this, the car would ram directly into the fire source.

📐 DISTANCE FORMULA

The ESP32 measures how long the ECHO pin stays HIGH (in microseconds).

distance = (pulse_duration × 0.0343) ÷ 2

0.0343 cm/µs = speed of sound at room temperature.
Divide by 2 because the pulse travels TO the object and BACK.

⚡ VOLTAGE DIVIDER (IMPORTANT!)

ESP32 GPIO pins are 3.3V. The HC-SR04 ECHO pin outputs 5V.

You MUST protect the ESP32 with a voltage divider:
ECHO → 1kΩ → GPIO19 → 2kΩ → GND

Or simply use the HC-SR04P variant which works at 3.3V natively.
fire_fighting_car_ultrasonic.py — MicroPython ESP32 + HC-SR04 obstacle avoidance & fire distance 
from machine import Pin, PWM, time_pulse_us
import time

# ── IR Sensors ──────────────────────────────
IR_LEFT   = Pin(34, Pin.IN)
IR_CENTER = Pin(35, Pin.IN)
IR_RIGHT  = Pin(32, Pin.IN)

# ── HC-SR04 Ultrasonic (NEW) ─────────────────
TRIG = Pin(18, Pin.OUT)   # Trigger
ECHO = Pin(19, Pin.IN)    # Echo (use voltage divider!)

# ── L298N Motor Driver ───────────────────────
IN1 = Pin(26, Pin.OUT);  IN2 = Pin(27, Pin.OUT)
IN3 = Pin(25, Pin.OUT);  IN4 = Pin(33, Pin.OUT)
ENA = PWM(Pin(14), freq=1000)
ENB = PWM(Pin(12), freq=1000)

# ── Pump + Servo ─────────────────────────────
PUMP_PIN  = Pin(13, Pin.OUT)
SERVO_PIN = PWM(Pin(15), freq=50)

# ── Distance Thresholds (cm) ─────────────────
OBSTACLE_STOP_CM  = 15   # Stop if wall is this close
FIRE_SPRAY_CM     = 30   # Start spraying at this distance
FIRE_TOO_CLOSE_CM = 10   # Emergency stop

# ── Ultrasonic: get distance ─────────────────
def get_distance_cm():
    TRIG.value(0); time.sleep_us(2)
    TRIG.value(1); time.sleep_us(10)
    TRIG.value(0)
    try:
        dur = time_pulse_us(ECHO, 1, 30000)
        return 999 if dur < 0 else (dur * 0.0343) / 2.0
    except: return 999

# ── Motor helpers ────────────────────────────
def move_forward(s=900):
    IN1.value(1);IN2.value(0);IN3.value(1);IN4.value(0)
    ENA.duty(s);ENB.duty(s)
def turn_left(s=700):
    IN1.value(0);IN2.value(1);IN3.value(1);IN4.value(0)
    ENA.duty(s);ENB.duty(s)
def turn_right(s=700):
    IN1.value(1);IN2.value(0);IN3.value(0);IN4.value(1)
    ENA.duty(s);ENB.duty(s)
def move_backward(s=600):
    IN1.value(0);IN2.value(1);IN3.value(0);IN4.value(1)
    ENA.duty(s);ENB.duty(s)
def stop_motors():
    ENA.duty(0);ENB.duty(0)

# ── Obstacle avoidance ───────────────────────
def avoid_obstacle():
    stop_motors(); time.sleep_ms(200)
    move_backward(); time.sleep_ms(500)
    turn_right();    time.sleep_ms(600)
    stop_motors()

# ── Approach fire (ultrasonic guided) ────────
def approach_fire():
    while True:
        dist = get_distance_cm()
        if dist <= FIRE_TOO_CLOSE_CM:
            stop_motors(); return True   # Too close!
        if dist <= FIRE_SPRAY_CM:
            stop_motors(); return True   # In range!
        if IR_CENTER.value() == 1:
            stop_motors(); return False  # Fire lost
        move_forward(450)               # Approach slowly
        time.sleep_ms(150)

# ── Fight fire ───────────────────────────────
def fight_fire():
    if not approach_fire(): return
    PUMP_PIN.value(1)
    for a in range(60,121,5): SERVO_PIN.duty(26+int(a/180*102)); time.sleep_ms(60)
    for a in range(120,59,-5): SERVO_PIN.duty(26+int(a/180*102)); time.sleep_ms(60)
    time.sleep_ms(2000)
    PUMP_PIN.value(0)
    move_backward(); time.sleep_ms(600)
    stop_motors()

# ── MAIN LOOP ────────────────────────────────
stop_motors(); PUMP_PIN.value(0)
while True:
    L = IR_LEFT.value()==0
    C = IR_CENTER.value()==0
    R = IR_RIGHT.value()==0

    if C:
        fight_fire()              # Fire ahead → approach + spray
    elif L and not R:
        turn_left(); time.sleep_ms(200)
    elif R and not L:
        turn_right(); time.sleep_ms(200)
    else:
        dist = get_distance_cm()  # Obstacle check while patrolling
        if dist < OBSTACLE_STOP_CM:
            avoid_obstacle()
        else:
            move_forward()

    time.sleep_ms(100)
